Python程序语法元素分析

2025.10.01 Python

引言

本单元将围绕经典的“温度转换实例”，系统地讲解Python编程中的核心语法元素。无论你是初次接触编程的新手，还是已有一定基础的学习者，都可以通过本节内容建立起对Python程序结构的清晰认知。

我们将逐一解析：

  • 程序的格式框架
  • 命名规则与保留字
  • 数据类型（整数、浮点数、字符串、列表）
  • 语句与函数的基本用法
  • 输入输出机制

最后，我们会对整个温度转换程序进行逐行代码分析，帮助你理解一个完整Python小程序的运行逻辑。

目标是：让你能够读懂并理解10行左右的Python程序。

一、程序的格式框架

我们先来看一段典型的Python代码——温度转换程序：

# 该程序用于实现华氏温度与摄氏温度之间的相互转换
TempStr = input("请输入带有符号的温度值: ")
if TempStr[-1] in ['F', 'f']:
    C = (eval(TempStr[0:-1]) - 32) / 1.8
    print("转换后的温度是{:.2f}C".format(C))
elif TempStr[-1] in ['C', 'c']:
    F = 1.8 * eval(TempStr[0:-1]) + 32
    print("转换后的温度是{:.2f}F".format(F))
else:
    print("输入格式错误")

1. 代码高亮（非语法要求）

  • 在编辑器中，不同部分会以不同颜色显示（如关键字蓝色、字符串绿色等）。
  • 这是代码高亮，由开发环境提供，用于提升可读性。
  • 注意：颜色不影响程序执行，仅是视觉辅助。

2. 缩进（Indentation）——Python的核心语法之一

  • 定义：一行代码前的空白区域（空格或Tab）。
  • 作用：表达程序的层次结构和包含关系。
  • 重要性：缩进是Python语法的一部分，错误的缩进会导致程序报错。

✅ 正确示例：

if condition:
    print("这是缩进块内的语句")  # 属于if分支

❌ 错误示例：

if condition:
print("缺少缩进！")  # 报错：Expected an indented block

缩进规则：

  • 同一级别的代码块必须使用相同数量的缩进。
  • 推荐使用 4个空格 作为一级缩进（PEP 8规范）。
  • 不建议混合使用空格和Tab，容易导致混乱。

在上述程序中：

  • if、elif、else 后的语句都有一层缩进，表示它们属于各自的条件分支。

3. 注释（Comments）

  • 定义：程序员写的说明性文字，不被计算机执行。
  • 作用：提高代码可读性和维护性。

Python支持两种注释方式：

类型 写法 示例
单行注释 # 开头 # 这是一个注释
多行注释 ''' ... ''' 或 """ ... """ 
'''
这是一个多行注释
可以跨越多行
'''

⚠️ 注意：虽然多行字符串可以用作文档字符串（docstring），但真正意义上的“多行注释”其实是多个单行注释更常见。

二、命名与保留字

1. 变量（Variable）

  • 定义：用于存储和表示数据的占位符。
  • 命名过程：为变量赋予一个名字，称为“命名”。

例如：

TempStr = "82F"
  • TempStr 是变量名，代表用户输入的温度字符串。

命名规则：

  1. 使用字母、数字、下划线 _ 和汉字等字符组合；
  2. 首字符不能是数字（如 123abc ❌）；
  3. 区分大小写（TempStr ≠ tempstr）；
  4. 不能与保留字重名。

✅ 合法命名示例：

  • my_var
  • temperature
  • 姓名
  • Python_课程

❌ 非法命名示例：

  • 1var （首字符为数字）
  • for （与保留字冲突）

2. 保留字（Reserved Words / Keywords）

  • 定义：被Python语言内部定义并保留使用的标识符，不能用作变量名。
  • 共有 33个 保留字。

Python保留字列表：

分类 关键词
逻辑运算 and, or, not
条件控制 if, elif, else
循环结构 for, while, break, continue
函数定义 def, return, lambda
异常处理 try, except, finally, raise
类与对象 class, self, super
模块导入 import, from, as
特殊值 True, False, None
其他 assert, del, global, nonlocal, pass, with, yield

🔹 本课程重点掌握其中 26个基础语法相关保留字。
🔹 True, False, None 首字母大写，是常量而非变量。

📌 在温度转换程序中出现的保留字：

  • if, elif, else：构成分支结构
  • in：判断成员是否存在于集合中

💡 小技巧：小写 if 是保留字；大写 If 可作为变量名（但不推荐）。

三、数据类型

Python中有多种数据类型。本程序涉及以下四种基本类型：

1. 字符串（String）

  • 定义：由零个或多个字符组成的有序序列。
  • 表示方法：用单引号 ' ' 或双引号 " " 括起来。
"Hello"
'Python'
"请输入带有符号的温度值"

字符串索引与切片

由于字符串是有序序列，可以通过索引访问单个字符。

有两种索引体系：

正向递增索引 0 1 2 3 4 5 …
字符 P y t h o n
反向递减索引 -6 -5 -4 -3 -2 -1

示例：

s = "Python"
print(s[0])   # 输出 'P'
print(s[-1])  # 输出 'n'

切片操作（Substring Extraction）

语法：[M:N] 表示从索引 M 到 N-1 的子串。

s = "Python"
print(s[1:4])  # 输出 'yth'（索引1~3）
print(s[0:-1]) # 输出 'Pytho'（去掉最后一个字符）

在程序中的应用：

TempStr[0:-1]  # 去掉最后一个字符（如"82F" → "82"）
TempStr[-1]    # 获取最后一个字符（如"F"）

2. 整数类型（int）

  • 对应数学中的整数，包括正负整数。
  • 示例：32, -10, 100
a = 32
b = -5

3. 浮点数类型（float）

  • 对应实数，带小数点的数字。
  • 示例：1.8, 3.14159, -0.5
pi = 3.14159
rate = 1.8

4. 列表类型（list）

  • 定义：由零个或多个元素组成的有序序列。
  • 表示方法：用方括号 [ ] 括起，元素间用逗号分隔。
['F', 'f']
['C', 'c']
[1, 2, 3]

成员检测：in 操作符

x in [1, 2, 3]  # 如果x在列表中返回True，否则False

在程序中的应用：

if TempStr[-1] in ['F', 'f']:
    # 如果最后一个字符是'F'或'f'，执行转换逻辑

四、语句与函数

1. 赋值语句（Assignment Statement）

  • 使用 =（赋值符号）将右侧表达式的值赋给左侧变量。
  • 语法：变量 = 表达式
TempStr = input("请输入...")
C = (eval(...) - 32) / 1.8

⚠️ 注意：= 是赋值，不是数学中的等于。比较相等使用 ==。

2. 分支语句（Conditional Statements）

根据条件决定执行哪段代码。

结构：

if 条件1:
    语句块1
elif 条件2:
    语句块2
else:
    语句块3

关键点：

  • 每个条件后必须加冒号 :；
  • 条件成立时执行对应缩进块；
  • elif 和 else 可选。

在程序中的体现：

if TempStr[-1] in ['F','f']:      # 判断是否为华氏度
    C = ...
elif TempStr[-1] in ['C','c']:    # 判断是否为摄氏度
    F = ...
else:                             # 其他情况
    print("输入格式错误")

3. 函数（Function）

  • 定义：封装了一段可重复使用的代码，接受输入参数并返回结果。
  • 类似于数学中的 $ y = f(x) $

函数调用语法：

函数名(参数)

程序中使用的三个关键函数：

函数 功能 示例
input() 从键盘获取用户输入 input("提示信息")
print() 向屏幕输出结果 print("hello")
eval() 执行字符串形式的Python表达式 eval("1+2") → 3

五、Python程序的输入输出

1. 输入函数：input()

  • 功能：从控制台接收用户输入的信息。
  • 返回值：字符串类型（即使输入的是数字）。
TempStr = input("请输入带有符号的温度值: ")
  • 用户输入 82F，则 TempStr = "82F"
  • 提示字符串 "请输入..." 不会被存入变量

📌 特别注意：所有输入默认都是字符串，若需数值运算，必须转换！

2. 输出函数：print()

  • 功能：将信息打印到屏幕。

基本用法：

print("转换后的温度是{:.2f}C".format(C))

格式化输出详解：

  • {}：称为“槽”（slot），表示待填充的位置；
  • .format(C)：将变量 C 的值填入第一个 {} 中；
  • :.2f：格式化为浮点数，保留两位小数。

示例：

C = 27.777777
print("结果是{:.2f}".format(C))  # 输出：结果是27.78

💡 记忆口诀：“大括号占位置，format来填充，:.2f取两位”

3. 评估函数：eval()

  • 功能：去掉字符串最外层引号，并执行其内容作为Python表达式。
  • 俗称：“求值函数”或“评估函数”

工作原理：

eval("1")        → 1           # 字符串转整数
eval("1+2")      → 3           # 执行加法运算
eval("'Hello'")  → "Hello"     # 去掉外层引号
eval("print('Hi')") → 执行print，输出 Hi

在程序中的应用：

TempStr = "82F"
num_str = TempStr[0:-1]   # 得到 "82"
num = eval(num_str)       # 得到 数字 82（可用于计算）

⚠️ 安全警告：eval() 执行任意代码，慎用于不可信输入！

六、温度转换程序逐行分析

现在我们结合以上知识，逐行解读这个程序：

# 1. 注释：说明程序用途，不被执行
# 该程序用于实现华氏温度与摄氏温度之间的相互转换

# 2. 输入：获取用户输入的温度字符串
TempStr = input("请输入带有符号的温度值: ")

# 3. 分支判断：检查是否为华氏温度（结尾为F或f）
if TempStr[-1] in ['F', 'f']:
    # 4. 提取数值部分（去掉最后一位），并用eval转为数字
    C = (eval(TempStr[0:-1]) - 32) / 1.8
    # 5. 输出转换后的摄氏温度，保留两位小数
    print("转换后的温度是{:.2f}C".format(C))

# 6. 否则如果结尾是C或c，视为摄氏温度
elif TempStr[-1] in ['C', 'c']:
    # 7. 提取数值并转换为华氏温度
    F = 1.8 * eval(TempStr[0:-1]) + 32
    # 8. 输出结果
    print("转换后的温度是{:.2f}F".format(F))

# 9. 其他情况：输入格式不符合预期
else:
    # 10. 提示错误
    print("输入格式错误")

✅ 程序运行流程图解：

开始
 ↓
输入温度字符串（如 "82F"）
 ↓
判断最后一个字符
 ├─ 是 'F'/'f' → 计算 C = (82 - 32)/1.8 ≈ 27.78 → 输出 "27.78C"
 ├─ 是 'C'/'c' → 计算 F = 1.8*数值 + 32 → 输出 "XX.XXF"
 └─ 否则 → 输出 "输入格式错误"
 ↓
结束
