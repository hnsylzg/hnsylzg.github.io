turtle库的使用

2025.10.01 Python

一、turtle库的基本介绍

在本模块中，我们将学习：

  • turtle库的基本概念
  • 绘图窗体的布局设置
  • 空间坐标体系与角度坐标体系
  • RGB色彩体系的基础知识

通过本单元的学习，你将掌握如何用Python控制一只“海龟”在屏幕上绘制图形，开启你的图形化编程之旅。

什么是turtle库？

turtle（中文意为“海龟”），也被称为海龟绘图系统（Turtle Graphics），是一种经典的图形绘制方法。它最早于1969年由Logo语言引入，专为程序设计初学者设计，强调“可视化编程”的理念。

🐢 想象一下：有一只小海龟，趴在一张白纸上，它的尾巴会留下痕迹。我们通过代码指挥这只海龟前进、后退、转弯……它走过的路径就形成了图形。

  • 地位：turtle是Python的标准库之一，无需额外安装，直接导入即可使用。
  • 用途：非常适合编程入门者理解循环、条件、函数等基本概念，并实现有趣的图形绘制。

💡 小知识：Python的标准库是随解释器一起安装的功能模块（如 math, random, turtle）；而第三方库需要手动安装（如 pygame, requests）。

虽然在专业术语中，“库”、“包”、“模块”有细微区别，但在本课程中，我们统一称为“模块”。

turtle的魅力

不要小看这只“小海龟”！它能绘制出非常复杂的图案。比如北京理工大学的学生们，仅用两周时间就创作出了令人惊叹的作品：

  • 几何艺术
  • 动画效果
  • 数据可视化雏形

你也可以做到！让我们从基础开始，一步步探索turtle的强大功能。

二、turtle绘图窗体布局

要开始绘图，首先需要一个“画布”——也就是一个独立的窗口。

这个窗口就是turtle的绘图区域，在操作系统中表现为一个图形窗口，最小单位是像素（pixel）。

例如：fd(100) 表示让海龟向前移动100个像素。

设置窗体：turtle.setup()

我们可以使用 setup() 函数来定义窗口的大小和位置。

语法：

turtle.setup(width, height, startx, starty)
参数 含义
width 窗口宽度（像素）
height 窗口高度（像素）
startx 窗口左上角在屏幕中的X坐标（可选）
starty 窗口左上角在屏幕中的Y坐标（可选）

⚠️ 注意：屏幕坐标系的原点 (0,0) 位于屏幕左上角，X向右增加，Y向下增加。

示例：

import turtle

# 示例1：窗口宽800，高400，左上角位于屏幕左上角
turtle.setup(800, 400, 0, 0)

# 示例2：只设置大小，位置由系统自动居中
turtle.setup(800, 400)

✅ 提示：setup() 不是必须的。如果不调用，系统会使用默认大小并居中显示窗口。

三、turtle空间坐标体系

在turtle的绘图窗口内部，有一个以画布中心为原点 (0, 0) 的坐标系。

1. 绝对坐标系（Fixed Coordinate System）

  • 原点 (0, 0) 在窗口正中心。
  • X轴向右为正，Y轴向上为正。
  • 整个窗口形成一个二维平面坐标系。

移动到指定坐标：goto(x, y)

turtle.goto(x, y)

将海龟从当前位置直接移动到坐标 (x, y)，并画出连线。

🐢 初始位置：程序启动时，海龟位于 (0, 0)，面向右侧（X轴正方向）。

示例：绘制一个正方形轮廓

import turtle

turtle.goto(100, 100)     # 右上角
turtle.goto(100, -100)    # 右下角
turtle.goto(-100, -100)   # 左下角
turtle.goto(-100, 100)    # 左上角
turtle.goto(0, 0)         # 回到原点

这段代码会画出一个边长为200像素的正方形边框。

2. 海龟坐标系（Relative Coordinate System）

这是以海龟自身为中心的相对坐标系，更符合“移动”的直觉。

方向 对应函数
前进 turtle.fd(distance) 或 forward()
后退 turtle.bk(distance) 或 backward()
左转 turtle.left(angle)
右转 turtle.right(angle)
画圆 turtle.circle(radius, extent=None)

🔄 circle() 是以海龟左侧某点为圆心，逆时针画弧。

这些函数基于海龟的当前朝向进行操作，非常适合绘制连续路径。

四、turtle角度坐标体系

海龟不仅有位置，还有朝向（方向）。

绝对角度坐标系

  • 0°：面向X轴正方向（向右）
  • 90°：面向Y轴正方向（向上）
  • 180°：面向X轴负方向（向左）
  • 270°：面向Y轴负方向（向下）

可以使用 seth() 函数设置海龟的绝对朝向。

语法：

turtle.seth(angle)

❗ seth() 只改变方向，不移动位置，也不绘制线条。

示例：

turtle.seth(45)      # 面向东北方向（45度）
turtle.fd(100)       # 向该方向前进100像素

turtle.seth(-135)    # 面向西南方向（-135度）
turtle.fd(100)

相对角度调整：left() 和 right()

除了绝对角度，还可以相对当前方向旋转：

turtle.left(90)   # 左转90度
turtle.right(45)  # 右转45度

实战示例：绘制Z形折线

目标：画一个斜向的“Z”字形。

import turtle

turtle.left(45)        # 左转45度（准备画第一斜线）
turtle.fd(150)         # 前进150像素

turtle.right(135)      # 右转135度（转向水平方向）
turtle.fd(300)         # 画中间横线

turtle.left(135)       # 左转135度（准备画最后一斜线）
turtle.fd(150)         # 完成Z形

turtle.done()  # 保持窗口打开（重要！）

✅ turtle.done()：防止程序结束后窗口立即关闭，确保你能看到绘图结果。

五、RGB色彩体系

为了让图形更生动，我们需要给线条和填充添加颜色。

什么是RGB？

RGB 是 Red（红）、Green（绿）、Blue（蓝） 三种颜色通道的组合，是计算机中最常用的色彩模型。

  • 通过调节三种颜色的强度，可以合成几乎所有可见颜色。
  • 每个通道的取值范围有两种模式：
模式 取值范围 示例（白色）
整数模式 0 ~ 255 (255, 255, 255)
小数模式 0.0 ~ 1.0 (1.0, 1.0, 1.0)

常见颜色对照表：

颜色 整数RGB 小数RGB
白色 (255,255,255) (1.0,1.0,1.0)
黑色 (0,0,0) (0.0,0.0,0.0)
红色 (255,0,0) (1.0,0.0,0.0)
绿色 (0,255,0) (0.0,1.0,0.0)
蓝色 (0,0,255) (0.0,0.0,1.0)
黄色 (255,255,0) (1.0,1.0,0.0)
紫色 (160,32,240) (0.63,0.13,0.94)

设置色彩模式：turtle.colormode()

turtle默认使用小数模式（0~1.0），但我们可以通过 colormode() 切换。

语法：

turtle.colormode(1.0)   # 使用小数模式
turtle.colormode(255)   # 使用整数模式

示例：设置画笔颜色

import turtle

# 方法1：使用小数模式
turtle.colormode(1.0)
turtle.pencolor(1.0, 0.0, 0.0)  # 红色

# 方法2：使用整数模式
turtle.colormode(255)
turtle.pencolor(255, 165, 0)    # 橙色

🎨 pencolor()：设置画笔颜色
🖍️ fillcolor()：设置填充颜色
🧱 color()：同时设置画笔和填充颜色
