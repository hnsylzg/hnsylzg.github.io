Arch Linux 安装与进阶配置指南

2026.01.05 Linux

一、 安装前的准备

1.1 开启 SSH 远程安装 (推荐)

在实机或虚拟机上，设置 root 密码并开启 SSH，以便从另一台电脑复制命令。

Bash

passwd
systemctl start sshd
ip addr  # 查看 IP 地址

1.2 键盘与时间同步

Bash

loadkeys us
timedatectl set-ntp true

二、 磁盘分区与格式化 (UEFI 标准)

2.1 分区方案 (GPT)

使用 fdisk /dev/sda 创建分区：

  • /dev/sda1: EFI 分区 (512MB~1GB)
  • /dev/sda2: Swap 分区 (建议内存的 0.5~1 倍)
  • /dev/sda3: 根目录 / (剩余全部)

2.2 格式化与挂载

Bash

mkfs.fat -F 32 /dev/sda1
mkswap /dev/sda2
swapon /dev/sda2
mkfs.ext4 /dev/sda3

mount /dev/sda3 /mnt
mount --mkdir /dev/sda1 /mnt/boot

三、 核心系统部署

3.1 安装基础包与必须的固件

注意： 必须包含 intel-ucode (Intel) 或 amd-ucode (AMD) 处理器微码。

Bash

pacstrap -K /mnt base linux linux-firmware base-devel networkmanager nano vi intel-ucode 
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt

四、 系统初始化配置

4.1 本地化设置

Bash

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
hwclock --systohc

# 编辑 /etc/locale.gen，取消 en_US.UTF-8 和 zh_CN.UTF-8 的注释
locale-gen
echo "LANG=en_US.UTF-8" > /etc/locale.conf

4.2 网络与用户权限

Bash

echo "archlinux" > /etc/hostname
systemctl enable NetworkManager
passwd  # 设置 root 密码

# 创建日常用户
useradd -m -G wheel,video,audio -s /bin/bash lzg
passwd lzg
# 编辑 sudoers (使用 visudo)
# 取消注释: %wheel ALL=(ALL:ALL) ALL

4.3 GRUB 引导 (含双系统检测)

Bash

pacman -S grub efibootmgr os-prober mtools dosfstools
# 如果是双系统，开启 os-prober
echo "GRUB_DISABLE_OS_PROBER=false" >> /etc/default/grub

# 自动保存菜单选项,菜单不显示子菜单
nano /etc/default/grub
GRUB_DISABLE_SUBMENU=y
GRUB_DEFAULT=saved
GRUB_SAVEDEFAULT=true

grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg

五、 桌面环境与驱动安装

5.1 显卡与音频

Bash

# 显卡驱动 (根据实际选择)
pacman -S xf86-video-intel      # Intel
# pacman -S nvidia-open # Nvidia
# 使用官方默认内核 (linux)	nvidia-open
# 使用多个内核（如 linux + linux-lts）	nvidia-open-dkms
# 使用自定义内核（如 linux-zen, linux-hardened）	nvidia-open-dkms

# 现代音频架构 PipeWire (替代 PulseAudio)
pacman -S pipewire pipewire-pulse pipewire-alsa pipewire-jack wireplumber

5.2 安装 KDE Plasma (或 Xfce)

Bash

pacman -S xorg-server sddm plasma-meta konsole dolphin ark gwenview
systemctl enable sddm

六、 进阶软件源与美化

6.1 配置 ArchLinuxCN 源与常用工具

编辑 /etc/pacman.conf，在末尾添加：

Ini, TOML

[archlinuxcn]
SigLevel = Optional TrustAll
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

安装必备组件：

Bash

pacman -Sy archlinuxcn-keyring
pacman -S yay 
# 安装中文字体与 WPS
yay -S noto-fonts-cjk noto-fonts-emoji wps-office-cn ttf-wps-fonts

6.2 Zsh 终端美化 (Manjaro 风格)

Bash

yay -S zsh manjaro-zsh-config
# 在 ~/.zshrc 中 source /usr/share/zsh/manjaro-zsh-config
chsh -s /bin/zsh

6.3 中文输入法 (Fcitx5)

Bash

pacman -S fcitx5-chinese-addons fcitx-configtool
# 编辑 /etc/environment
echo -e "GTK_IM_MODULE=fcitx\nQT_IM_MODULE=fcitx\nXMODIFIERS=@im=fcitx" >> /etc/environment

七、 虚拟机与系统维护 (必须)

7.1 VMware 优化

Bash

pacman -S open-vm-tools gtkmm3
systemctl enable --now vmtoolsd
# 解决分辨率问题：在 /etc/mkinitcpio.conf 的 MODULES 加上 (vmwgfx)
mkinitcpio -P

7.2 SSD 维护

Bash

systemctl enable fstrim.timer

7.3 清理孤立包 (定期执行)

Bash

pacman -Qtdq | sudo pacman -Rns -

安装完成，现在可以执行 reboot 重启进入新系统了！
